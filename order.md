あなたはデザイン実装に強い熟練したフロントエンドエンジニアです。

私のWebサイト（自家焙煎珈琲と陶器の店舗LP）のメインビジュアル（ヒーローセクション）の実装をお願いします。



## 1. 実装のゴール

福岡女学院大学「感性メディア学科」特設サイト（https://www.fukujo.ac.jp/university/kansei-media/）のアニメーション挙動と、添付画像のレイアウト構成（対角線上の文字配置）を再現し、**「陶器と炭火」**の世界観を表現したLPを作成してください。



## 2. デザイン・レイアウト構成（添付画像参照）

画面全体をフルスクリーン（100vh）で使用し、以下の要素を配置してください。すべて**縦書き（vertical-rl）**を基本とします。



**A. メインタイトル（対角配置）**

   - **右上**: 「陶器、」

     - 大きく配置。フォントは明朝体または筆文字風。

     - 色: `--accent-color` (グラデーションも可)

   - **左下**: 「焙煎」

     - 「陶器、」と対になるように配置。

     - 色: `--text-color` (白系)



**B. サブコピー（中央配置）**

   - **画面中央**: 「栃木県栃木市にて、陶器と炭火が紡ぐ珈琲。」

     - 白い長方形の枠線（border）や、背景にうっすらとした白い座布団を敷き、縦書きで配置。

     - 文字サイズは小さめで上品に。



**C. ナビゲーション（下部中央）**

   - 画面下部中央に、小さな横並びのメニューリンクを配置。

   - [私たちのこと] [商品一覧] [お問い合わせ] 



**D. 背景**

   - フルスクリーンの画像または動画。

   - 全体に暗めのオーバーレイをかける。



## 3. 配色設定（和モダン・陶器・炭火）

以下のCSS変数 (:root) を使用してください。

- `--bg-color`: #1a1614;  /* 鉄黒：背景ベース */

- `--text-color`: #f2f0eb; /* 生成り：メインテキスト */

- `--accent-color`: #d2691e; /* 琥珀色：アクセント（陶器、の部分に使用） */

- `--overlay`: linear-gradient(to bottom, rgba(26,22,20,0.3), rgba(26,22,20,0.7));



## 4. アニメーション要件 (GSAP & ScrollTrigger)

**「ひらけ、感性」のサイト同様のオープニング演出**

1. **初期状態**:

   - 画面全体が黒（または暗色）のマスクで覆われている。

   - 中央に細いスリット（隙間）だけが見えている状態。

2. **ロード時**:

   - GSAPで `clip-path: inset(...)` をアニメーションさせ、マスクが中央から上下（または左右）にゆっくりと開き、背景画像が現れる。

   - 背景が開いた後、Aの「陶器、」「焙煎」と、Bの「サブコピー」が、ふわっと（opacity: 0 -> 1, blur: 10px -> 0px）出現する。

3. **イージング**:

   - `Power3.out` を使用し、**3.0秒** かけて重厚かつ滑らかに動かす。
