# 工藝 迎ふ (Kogei Mukau) - 炭火焙煎珈琲

「工藝 迎ふ」のブランドサイトです。

## コンセプト

「陶器」と「炭火」が紡ぐ特別な珈琲。

独自開発の陶器製焙煎機と、おが備長炭による炭火焙煎で、
雑味のない澄んだ味わいのスペシャルティコーヒーをお届けしています。

## 特徴

- **陶器製焙煎機**: 独自開発の焙煎機による熱伝達効果
- **おが備長炭**: 炭火の遠赤外線効果で豆を芯からふくらませる
- **スペシャルティコーヒー**: Qグレード認証など高品質な豆を厳選

## デザインシステム

宮崎製作所のデザインをベースに、「職人技・信頼感・和モダン」を表現。

- **カラーパレット**: 炭色の黒、陶器の茶褐色、和紙のような白
- **タイポグラフィ**: 明朝体（見出し）とゴシック体（本文）
- **アニメーション**: GSAP + ScrollTriggerによる高品質なスクロール演出

## 技術スタック

- HTML5
- Tailwind CSS (CDN)
- Google Fonts (Noto Sans JP / Noto Serif JP)
- GSAP 3.12.5 + ScrollTrigger

## ローカル開発

```bash
# ローカルサーバーを起動
python3 -m http.server 8000

# ブラウザで開く
open http://localhost:8000/index.html
```
